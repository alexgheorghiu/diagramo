Diagramo architecture explained


Diagramo is made out of :

  1. JavaScript engine: made out of Math(geometric) library and UI library
  2. PHP engine
  3. mySQL and file system part


JavaScript engine

JavaScript is the main programming language used in Diagramo. Combined with HTML5' canvas specification creates the core of the math library.
There are 2 sections to the JavaScript Engine:

  1. Math library

  2. UI library

Math Library - is a set of objects and functions that lay the foundation for any figure in the application. It contains:

  • primitives.js - contains definitions for Point, Line, Polygon, Arc, QuadCurve, CubicCurve, etc
  • matrix.js - contains operations you need for matrixes
  • text.js - contains the Text class needed to draw a text on canvas
  • util.js - a collection of functions needed by other modules (including primitives)
